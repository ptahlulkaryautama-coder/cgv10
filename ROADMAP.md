# CGV10 Roadmap

Approved public identity:

CGV10 — Portal Digital Warga  
Cipta Greenville • RT 010 / RW 021

CGV10 is a phased resident portal product. The current website represents Phase 1 only: a public resident portal foundation with PALUGADA CGV as a priority campaign.

## Official 8-Phase Roadmap

1. Phase 1 — Public portal + PALUGADA campaign
2. Phase 2 — Login warga + profil rumah
3. Phase 3 — Community feed
4. Phase 4 — PALUGADA katalog
5. Phase 5 — Keluhan, lapor RT, security
6. Phase 6 — Group chat basic
7. Phase 7 — Push notification + moderation
8. Phase 8 — Full resident app

## Phase 1 Focus

Phase 1 keeps the scope public and static:

- Public resident information
- Pengurus information structure
- Keuangan / kas transparency structure
- Pengumuman structure
- Kontak penting structure
- PALUGADA CGV priority campaign and catalogue preview

Phase 1 is not the final full app. It does not include login, database, payment, cart, transaction system, admin dashboard, real complaint system, group chat, push notification, real QR, real WA integration, or active PALUGADA transactions.

## PALUGADA CGV Direction

PALUGADA CGV should be presented as:

- Marketplace Warga
- A resident promotion and directory space
- A priority campaign from the RT direction
- A catalogue preview that can later grow into a fuller PALUGADA katalog phase

Until official item data is approved, PALUGADA content must remain clearly marked as preview/example content and should not look like an active transaction system.

## Content Safety

Do not mention any city name. Do not invent names, financial values, phone numbers, QR codes, WA links, or active PALUGADA item data.
