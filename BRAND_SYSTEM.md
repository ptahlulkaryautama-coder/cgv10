# CGV10 Brand System

## Brand Architecture

CGV10 is the parent brand and main resident portal.

PALUGADA CGV is a sub-brand for marketplace warga activity inside the CGV10 ecosystem.

Cipta Greenville and RT 010 / RW 021 are identity badges. They support trust and context, but they should not be merged into the main CGV10 logo lockup.

Feature icons are a separate UI icon system. They should not be embedded inside the CGV10 or PALUGADA CGV logo.

## Logo Usage

The complex logo concepts from pengurus are reference and emblem concepts only. They should not be used as one small website header logo because detailed text and emblem shapes become unreadable at navigation size.

The website header now uses a header-friendly proposal-based CGV10 mark:

- `public/assets/brand/cgv10-header-mark.svg`

This mark is derived from the CGV10 proposal direction, but simplified for navigation-size readability. It should remain paired with the text lockup:

- CGV10
- Portal Digital Warga

Supporting simplified assets remain available for future replacements or layout variants:

- `public/assets/brand/cgv10-mark.svg`
- `public/assets/brand/cgv10-wordmark.svg`

Identity context should stay separate from the main header logo:

- Cipta Greenville identity badge
- RT 010 / RW 021 identity badge
- Future official logo slot

PALUGADA CGV should use simplified sub-brand assets:

- `public/assets/brand/palugada-mark.svg`
- `public/assets/brand/palugada-wordmark.svg`

## Icon System

CGV10 feature icons:

- Warga
- Layanan
- Keuangan
- Informasi
- Keamanan

PALUGADA CGV category icons:

- Barang
- Kuliner
- Jasa
- Properti
- Lainnya

Icons should stay flat, simple, scalable, and readable at small sizes. Avoid 3D effects, fake official seals, tiny unreadable text, or mixing every service into one logo.

## Replacement Rule

Final approved logo files can replace the temporary SVG proposal system later. Keep filenames stable where possible so the UI does not need code changes. Do not publish unapproved private assets or complex JPG concepts as production header logos.
